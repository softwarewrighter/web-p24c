Build a browser-based Yew/Rust/WASM Pascal Compiler IDE for COR24 p-code VM. Read-only Pascal source and pre-compiled p-code assembly side by side, with link/assemble/run entirely in WASM. Phase 1: scaffold Yew project with Trunk, wire cor24-emulator + pa24r + pl24r dependencies, build.rs to pre-assemble pvm.s. Phase 2: demo registry — embed .pas + .spc pairs via include_str!(), demo selector dropdown. Phase 3: UI layout — four-panel grid (Pascal source, p-code assembly, output, hardware), Catppuccin Mocha theme. Phase 4: Link & Run pipeline — pl24r linking, pa24r assembly, loader (load .p24 at 0x010000, relocate data refs), batch execution. Phase 5: I/O and hardware — UART output display, UART input field, LED D2 indicator, S2 switch toggle. Phase 6: status bar + polish — Ready/Running/Exited states, instruction count, binary size, Stop button.
